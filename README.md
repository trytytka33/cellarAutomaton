# How to use
## Install dependencies
- sfml 3.0+
- cmake 3.10+
- cpp 17
## Setup cmake
- create ```CMakeLists.txt``` in project root
- setup it for your environment
- you can check example file
## Setup build & how to run
- create ```build``` dir
- go into ```build``` dir
- ```cmake ..``` 
- ```make``` 
- execute executable
