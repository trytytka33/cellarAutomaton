# LAB - 4 - Automaty Komórkowe 2D

# Zadania

Implementacja automatu komórkowego 2D symulującego "grę w życie" wraz z 4 podstawowymi regułami, ponadto:

* Implementacja 4 podstawowych stanów początkowych: Glider, Niezmienny, 2 inne (1 dowolny, 1 Oscylator z min. 15 komórek) (8 pkt)
* Implementacja warunku granicznego - periodycznego (4 pkt)
* Implementacja warunku granicznego - odbijającego (4 pkt)
* Wprowadzenie własnej reguły i ocena rezultatu (4 pkt)

# Ocena

* Maksymalna liczba punktów: 20
* Maksymalne dodatkowe punkty za kreatywne podejście do rozwiązania problemów / dodatkowe rozwiązania: 5 pkt

# Adnotacje

* Działający przykład online: <https://playgameoflife.com/>
* Przykłady stanów początkowych: <https://conwaylife.appspot.com/library/>
* Minamlna liczba przejść do zasymulowania: 30 :)
* Minimalny rozmiar przestrzeni: 200x200
* Powinna być zapewniona wizualizacja (np. w formie GUI) z każdego kroku symulacji (alternatywnie można wygenerować GIFa bądź film)
* Proponowanego dodatkowe zadania za punkty: zastosowanie znacznie większych wzorców, zmiana sąsiedztwa?
